VTproxy

Save your VirusTotal search quota by accessing VT results pages directly or by using VT API

Supported formats:

  • File hash: MD5, SHA-1, or SHA-256
  • URL: http:// or https://
  • Domain: example.com
  • IP: IPv4 or IPv6 address

VTproxy runs entirely in your browser. No data is sent to our servers - your searches and API key are processed locally and only communicate directly with VirusTotal.

API Key Settings

Enter your VirusTotal API key. Get one at virustotal.com/gui/my-apikey

Your API key is stored locally in your browser and never sent anywhere except VirusTotal.
